CASE SIZE ASSUMPTION — 40 UNITS/CASE

July Cases is calculated using a 40-unit case size for GA, based on the order data available (Flower 3.5g and 0.5g Vapes both ordered in 40-unit cases). This differs from CT's 30-unit case standard. Flag if case size varies by category or if this assumption needs correcting.

UNIT PRICE FALLBACKS ARE OBSERVED, NOT OFFICIAL LIST PRICES

Since GA has no established price list yet, the fallback prices used when a category has zero units ($22.50 for 3.5g Flower, $20.00 for 0.5g Vapes, sourced from actual wholesale order data) are placeholders. 1g Disposables (AIO) has no fallback price set (defaults to $0) since no pricing exists yet — update once available.

NO DEMAND PLAN — ALLOCATION-ONLY FOR NOW

This tracker currently shows actual units and revenue delivered only, with no attainment percentage or target comparison, since GA has no demand plan.
